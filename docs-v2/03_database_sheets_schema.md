## 1. ພາບລວມ / Overview
Google Sheets = ຖານຂໍ້ມູນ (DB). ແຕ່ລະ tab = 1 table. Row 1 = headers. ການເຂົ້າເຖິງທັງໝົດຜ່ານ `SheetRepository` ເທົ່ານັ້ນ (ບໍ່ເອີ້ນ `SpreadsheetApp` ໂດຍກົງໃນ service).

The active spreadsheet is resolved from Script Property `SHEET_ID`. All 11 tabs are auto-created by `setupDatabase()`.

## 2. ID strategy
Pattern `PREFIX-000N` (zero-pad 4). Generated by `nextId(entity, existingIds)` in `core/Ids.gs`.

| Entity | Prefix | Example |
|---|---|---|
| Customers | `CUS` | CUS-0001 |
| Albums | `ALB` | ALB-0001 |
| Photos | `PHO` | PHO-0001 |
| Documents | `DOC` | DOC-0001 |
| Orders | `ORD` | ORD-0001 |
| OrderItems | `ITM` | ITM-0001 |
| Payments | `PAY` | PAY-0001 |
| PrintJobs | `PRT` | PRT-0001 |
| Users | `USR` | USR-0001 |

## 3. Tables (11)

### Customers
`customer_id` (PK) · `name` * · `phone` · `email` · `address` · `birthday` · `tags` (CSV) · `created_at`

### Albums
`album_id` (PK) · `name` * · `customer_id` (FK) · `cover_photo_id` (FK) · `status` (active/archived) · `created_at`

### Photos
`photo_id` (PK) · `album_id` (FK) · `customer_id` (FK) · `name` * · `drive_file_id` · `sha256` (dedup key) · `size` · `favorite` (bool) · `status` (active/deleted) · `created_at`

### Documents
`document_id` (PK) · `name` * · `type` (PDF/DOCX/…) · `category` · `drive_file_id` · `version` (int) · `status` · `history` (JSON) · `created_at`

### Orders
`order_id` (PK) · `customer_id` * (FK) · `type` * · `status` (new→in_progress→printing→delivery→complete / cancelled) · `total` · `note` · `created_at`

### OrderItems
`item_id` (PK) · `order_id` (FK) · `description` · `qty` · `unit_price` · `amount`

### Payments
`payment_id` (PK) · `order_id` * (FK) · `customer_id` (FK) · `amount` * · `method` * (cash/transfer/qr) · `status` (paid) · `created_at`

### PrintJobs
`print_id` (PK) · `order_id` * (FK) · `type` * (Passport/Wedding/Certificate/Frame/Canvas/ID Card) · `size` · `qty` · `status` (queued→printing→done→reprint / cancelled) · `assigned_to` · `created_at`

### Users
`user_id` (PK) · `email` * (login key) · `name` · `role` * (Owner/Admin/Manager/Editor/Staff/Viewer) · `active` (bool)

### Settings
`key` (PK) · `value`

### Logs
`log_id` · `at` · `user` · `action` · `message`

## 4. Relationships
```
Customer 1─* Album 1─* Photo
Customer 1─* Order 1─* OrderItem
Order 1─* Payment
Order 1─* PrintJob
```

## 5. Validation (core/Validator.gs)
- `requireFields(obj, [...])` → throws `E_VALIDATION`
- `isEmail`, `isPhone` (≥6 digits)
- Order/Print status transitions enforced by state maps
- Payment method ∈ {cash, transfer, qr}; amount > 0

## 6. Setup / migration
```
setupDatabase()   // create 11 tabs + seed running user as Owner
setupDriveRoot()  // create shared Drive root, store ROOT_FOLDER_ID
validateSchema()  // { ok:true, missing:[] }
```
